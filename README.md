# Playwright-Tracewatch                                                                                                                                                                                                                                                                                                                                                                                      

▎ Selector health observability for Playwright test suites. Tracks locator stability over CI runs, surfaces at-risk selectors before they break your pipeline.
                                                                                                                                                                                                                                                                                                                                                                                           
### Why                                                                                                                                                                                                                                                                                                                                                                                      
Selector rot is one of the leading causes of flaky tests. Most teams find out a selector broke when CI fails — Tracewatch finds it before that.                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                                                         
### How it works                                                                                                                                                                                                                                                                                                                                                                             
1. Parses Playwright trace files after each CI run                                                                                                                                                                                                                                                                                                                                       
2. Tracks selector stability over time                                                                                                                                                                                                                                                                                                                                                   
3. Scores each locator's health across N runs
4. Surfaces degrading selectors in PR comments + suggests fixes via Claude API
